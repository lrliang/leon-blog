- 找不到合适工具的无奈

## 为什么选择桌面应用
- 专注沉浸的操作体验
- 数据本地存储，隐私优先
- 不依赖网络，随时可用

## 为什么选择 Rust
- 性能：桌面应用的天然要求
- 安全：内存安全保证应用稳定
- 学习：一直想学的语言，借项目驱动
- 生态：Dioxus 的成熟度让桌面开发可行

## 为什么选择 Dioxus
- React-like 的开发体验
- 纯 Rust 实现
- 跨平台能力
- 活跃的社区

## 这个项目要做什么
- 核心功能介绍
- MVP 范围定义
- 技术学习目标

## 接下来的计划

- 读者可以期待什么
