## 🎯 核心业务方向

### 业务一：企业咨询服务

**定位：** AI 赋能企业级软件开发效能

**价值主张：**
- 帮助企业团队提升 AI 辅助开发效率
- 软件架构咨询与技术方案设计
- 开发流程优化与团队效能提升

**目标客户：**
- 中小型软件企业 CTO/技术负责人
- 传统企业数字化转型团队
- 创业公司技术合伙人

### 业务二：个人应用销售

**定位：** 高质量工具类桌面应用

**价值主张：**
- 解决特定场景痛点的精品工具
- 本地优先、隐私安全
- 一次购买、永久使用

**首发产品：** asset-light（个人资产管理）

## 📱 自媒体矩阵

| 平台 | 定位 | 内容类型 | 优先级 |
|------|------|---------|--------|
| **微信公众号** | 深度输出、品牌沉淀 | 技术文章、案例分析、方法论 | P0 |
| **个人博客** | 技术积累、SEO 长尾 | 教程、开发日志、项目文档 | P0 |
| **掘金** | 技术社区、流量获取 | 技术教程、项目实战 | P0 |
| **小红书** | 触达泛人群、轻量传播 | 开发日常、工具推荐、效率技巧 | P1 |
| **抖音/视频号** | 视频引流、扩大影响 | 开发 Vlog、技术科普、工具演示 | P2 |

## 📅 关键里程碑

| 阶段 | 时间 | 目标 |
|------|------|------|
| **启动期** | 第 1-2 周 | 完成平台注册、基础人设、首批内容 |
| **冷启动期** | 第 3-8 周 | 持续输出、积累种子用户、建立内容节奏 |
| **增长期** | 第 2-3 月 | 视频矩阵启动、跨平台引流、首次转化 |
| **稳定期** | 第 4 月+ | 内容体系成熟、稳定获客、业务闭环 |
